# README #

Heritage project: a system for rule-based error detection

Пререквизиты:
	Tomita-pareser
	Все файлы должны лежать в той же папке, где лежит tomitaparser.exe.

Настройка:
В файле config.proto указать свои настройки:

	- Путь к входному файлу (в формате .txt);
	- Название и формат выходного файла (запись возможна в двух форматах -.txt/.json);
	
Использование:
	Выполнить ..\your path\tomitaparser.exe proto.config  
	Обработанный файл должен появиться в той же папке.
	Отладочный вывод можно посмотреть открыв через браузер файл pretty.html.
